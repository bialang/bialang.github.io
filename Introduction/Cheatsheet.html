    The Engine

    A Bia engine can be initialized like this:

    1
2
3
    #include <bia/bia.hpp>

bia::engine engine;

// always free afterwards
bia_engine_free(engine);

    Executing a Script

    The bia::engine::execute function takes a std::istream for the script:

    1
2
3
4
    #include <fstream>

std::ifstream file{ "myscript.bia" };
engine.execute(file);
